# Trivia Battle
A 2 player game of trivia questions made in React using Hooks and Redux. The players select between 4 - 24 topics and have 5 turns to make as most points possible. Try it with someone and see who the wisest player is!

## Built with:

* React (Hooks/Redux)
* CSS
* HTML

## APIS used:

* Open Trivia DB
